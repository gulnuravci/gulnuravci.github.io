About

A game resembling the Olympic Games built on Unity Game Engine in C#. The artwork for the game was done by me using SketchUp and Blender modeling software. Process included learning design concepts, storyboarding, and preparing documentation. Game incorporates educational athletic knowledge with quizzes implemented in the game.
